# PetClinic Functional Tests

Playwright-based test suite for the Spring PetClinic application.

## Quick Start

1. Install dependencies:
   ```powershell
   npm install
   ```
2. Install browsers:
   ```powershell
   npx playwright install
   ```
3. Run tests on Target Environment:
   ```powershell
   npx playwright test --project="Target Environment"
   ```

Refer to the [Root README](../README.md) for detailed instructions on multi-environment execution and configuration.

Detailed test case documentation can be found in the [testcases/](./testcases/README.md) folder.
